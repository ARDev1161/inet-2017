Работы

Экосистема разработки программ с открытым кодом

Целью данной лабораторной работы является создание собственной персональной страницы-отчёта в github

>> <

  • Шаг 1. Зайти в репозиторий stankin/inet-2017 и создать собственный форк.

  • Шаг 2. Клонировать репозиторий на локальный компьютер. Для этого необходимо использовать утилиту git bash.

  • Шаг 3. В программе git bash ввести команду git clone, указав скопированную ссылку в качестве аргумента.

  • Шаг 4. Создать файл индивидуальной страницы. Для этого необходимо создать папку, названную в виде "Фамилия_И_О". Внутри данной папки необходимо создать файл index.html, заполнив его стандартной структурой html файла.

  • Шаг 5. В программе git bash выполнить команду git add для добавления файлов под версионный контроль. После этого необходимо выполнить команду git commit -m "", дописав в кавычках информацию о коммите.

  • Шаг 6. Для публикации коммита в основном репозитории необходимо выполнить команду git push origin master.

  • Шаг 7. Затем необходимо предложить коммит для внесения в основной проект. Для этого в репозитории GitHub необходимо нажать кнопку Pull Request.

Описание

Приложение решает краевую задачу для уравнения теплопроводности в одномерном пространстве.

Требования к программе

  1. Создать интерактивную форму для отображения приложения.
  2. На форму добавить поле "Начальные условия", в дальнейшем здесь будут вводиться формулы начальных условий для всех точек.
  3. Создать два поля ввода формул краевых точек (аналогично предыдущему пункту).
  4. Добавить поле для ввода формулы, которая вносит условия для изменения температуры на точках.
  5. Создать отдельный блок для ввода следующих данных: длина, наблюдаемое время, делители длины и времени
  6. Добавить кнопку "Рассчитать"
  7. По окончании вычислений строится трёхмерный температурный график
