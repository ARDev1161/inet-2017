  • Лабораторная работа №2 "Разработка простого веб-приложения":

    Проектирование и разработка индивидуального или коллективного веб-приложения (SPA) с использованием html,css,js + json,xml

    Требования к SPA приложению:

    1. Приложение должно представлять из себя хорошо оформленную базу данных с различными блюдами, их описанием, ингредиентами и пошаговым рецептом приготовления.

    2. В приложении должен осуществляться фильтр по национальным блюдам, ингредиентам и типам блюда.

    3. При нажатии на блок с блюдом должно открываться описание блюда, его изображение, ингредиенты и пошаговый рецепт с картинками.

    4. Вверху окна приложения должен быть закреплённый банер, при нажатии на который происходит переход на начало страницы.

    5. Слева окна приложения должен быть закреплённый банер с параметрами выбора фильтров по кухне, ингредиентам и типу блюда.

    Описание. Результат: SPA приложение.

    ЗАДАЧИ:

    1) Собрать команду для разработки веб-приложения (SPA) и распределить роли. Страница команд.

    2) Сделать разметку страницы по макету. Описание приложения. Макет страницы. Исходный код на index.html.

    3) Прописать стили страницы. Исходный код на styles.css.

    4) Прописать JavaScript функционал и реализовать возможность чтения json:

    4.1) Разработать json файлы с рецептами, ингридиентами и типами блюд. Исходный код на products.json, ingrid.json, country.json, type.json, forwhat.json.

    4.2) Реализовать чтение из json файлов в список фильтров. Исходный код на script2.js.

    4.3) Разработать фильтрацию списка рецептов. Исходный код на script.js.

    4.4) Реализовать просмотр отдельных рецептов. Исходный код на script.js.

    Итог: SPA приложение. Страница репозитория. Исходный код на index.html, styles.css, script.js, script2.js, products.json, ingrid.json, country.json, type.json, forwhat.json.
