Подготовка к экзамену

Самооценка прохождения теста по инновациям, формулировка одного вопроса по теме "Интернет-технологии" в одной из форм, встречающихся в тесте (выбор одного, выбор многих, упорядочивание, соответствие)

Вопрос:

Ниже описаны уровни модели OSI и их протоколы. Поставьте соответствия между уровнями и их протоколами

Уровень Протокол
1) Прикладной A) PPP (Point-to-Point Protocol)
2) Представления данныхB) Telnet (teletype network)
3) СеансовыйC) OSPF (Open Shortest Path First)
4) ТранспортныйD) IP (Internet Protocol)
5) СетевойE) SSL (Secure Sockets Layer)
6) КанальныйF) 802.11 Wi-Fi
7) ФизическийG) HTTP (HyperText Transfer Protocol
- H) Token Ring
- I) UDP (User Datagram Protocol)
- J) PPTP (Point-to-Point Tunneling Protocol)
- K) TCP (Transmission Control Protocol)
- L) SMTP (Simple Mail Transfer Protocol)

Ответы:
1)B, G, L
2)E
3)J
4)I, K
5)C, D
6)A, H
7)F

Благодаря этому вопросу проверяются знания о протоколах в сетевой базовой эталонной модели OSI. Данные знания необходимы для понимания того, какие протоколы работают на определённых уровнях обмена данными в сети.

Вопрос сформулирован в форме «соответствие» для проверки знаний о протоколах модели OSI, чтобы определить насколько хорошо студент разбирается в протоколах каждого уровня.
